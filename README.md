# Ticket Reservation App

Ticket reservation application.

Current status:

- Vue.js client is in progress.
- Backend services are not implemented yet.

## Client

```bash
cd client
npm.cmd install
npm.cmd run dev
```

Development URL:

```text
http://localhost:5173
```

## Development Checks

```bash
cd client
npm.cmd run lint
npm.cmd run build
```
